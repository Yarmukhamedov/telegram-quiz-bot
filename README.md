# 🤖 Telegram Quiz Bot (v2.0)

Бот для проверки знаний в программировании. Теперь с поддержкой нативных опросов Telegram и случайным порядком вопросов.

## ✨ Новые возможности:
- **20 вопросов** по темам: Терминал, IDE, Python, Git, HTML и CSS.
- **Рандомизация:** Каждый раз вопросы перемешиваются — никакой скуки!
- **Нативные опросы:** Анимация, конфетти и встроенные подсказки.

## 🚀 Быстрый старт (PythonAnywhere)

1.  **Обнови код**:
    ```bash
    cd название-репозитория
    git pull origin main
    ```
2.  **Перезапусти**:
    - Останови старый процесс (`Ctrl+C`).
    - Запусти: `python3 bot.py`.

## 📚 Как добавлять вопросы?

Редактируй `questions.py`. Бот сам подхватит новые вопросы и добавит их в случайную выборку при следующем запуске `/quiz`.
